# ⚡ Voltage Regulator Circuit: Design & Verification
**Semester 1 Final Project | University of Central Punjab (UCP)**

### 📖 Overview
This project involves the mounting and verification of a **stable 5V DC power supply** from a 12V AC source. It demonstrates the transition from theoretical circuit design to physical hardware testing.

### 📊 Performance Results
- **Theoretical Output:** 5.0 V DC
- **Experimental Result:** 4.87 V DC (Verified via Digital Multimeter)
- **Software Simulation:** Verified using **Circuit Maker / MultiSim 11.0**

### 🛠️ Hardware Stack
- **IC Regulator:** L7805CV
- **Rectifier:** 4x IN4007 Diodes (Bridge Rectifier)
- **Filter:** 1000µF, 35V Capacitor
- **Tools:** Digital Multimeter (DMM), Oscilloscope, Breadboard
